# grdData
Formatação de data em dataGridView considerando as diversas culturas  mundiais

Exemplos de diversos modos de formatação de datas ( dia/mes/ano, mes/dia/ano  por exemplo) considerando a regionalização selecionada pelo usuário.

Escolhida uma data, hora e região o programa apresenta diversas possibilidades de formatação pré definidas. 

Inclui linguagens europeias, asiaticas, orientais etc.
